PART 03

大家在聊什么？

我翻了翻这73年的"群聊记录"，发现话题分了三期。

每个时期，人们关心的关键词都在变。

天仙配
1950s · 经典
《天仙配》
“树上的鸟儿成双对，绿水青山带笑颜。”董永与七仙女的爱情故事，是黄梅戏最广为人知的代表作。1955年拍摄成电影，轰动全国。
# 爱情 # 神话
女驸马
1960s · 传奇
《女驸马》
“为救李郎离家园，谁料皇榜中状元。”冯素珍女扮男装、冒名赶考的故事，体现了古代女性的智慧与勇气。
# 女扮男装 # 状元
打猪草
1970s · 田园
《打猪草》
“郎对花姐对花，一对对到田埂下。”一出充满乡土气息的小戏，表现劳动人民淳朴的爱情与生活。
# 田园 # 对花
徽州女人
1990s · 现代
《徽州女人》
以徽州文化为背景，讲述女人的等待与坚韧。从传统走向现代，探索黄梅戏的更多可能，曾获得文华大奖。
# 新编 # 女性
🌐
新国潮
2020s · 融合
黄梅戏 · 新国潮
从 B 站到直播间，黄梅戏正在年轻化。Rap、电音、短视频元素进入唱腔和传播，让老戏腔继续生长。
# 创新 # 元宇宙
严凤英
黄梅戏之魂
严凤英 (1930-1968)
黄梅戏一代宗师，嗓音甜美、表演真挚，将黄梅戏推向全国。代表作包括《天仙配》《女驸马》《牛郎织女》。
# 致敬 # 大师
"危机""流失"等焦虑词汇在2010年后出现约 67 次
